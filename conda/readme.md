## conda commands


#### conda create env

```bash
conda create -p ./envs python=3.10 -y
```
or (if not create env in current path)
```bash
conda create -n envs python=3.10 -y
```
</br>

#### conda activate env

```bash
conda activate envs/
```
or (if not create env in current path)
```bash
conda activate envs
```
</br>

#### conda list env

```bash
conda env list
```
</br>
